# Summary

This clip provides secure mounting for the LifeCam HD-3000 camera.    You'll need to remove the floppy rubber-covered bracket by either opening up the camera and fiddling with impossibly tiny screws, or do like I did and cut the thing off with a Dremel abrasive cut-off wheel.

I've also included a version with a mounting surface for a 60mm dia light ring.
